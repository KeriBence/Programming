Python listák

A listák több elem tárolására szolgálnak egyetlen változóban.

A listák a Pythonban az adatgyűjtemények tárolására használt 4 beépített adattípus egyike, a másik 3 pedig a Tuple, Set és Dictionary, amelyek mindegyike eltérő minőségű és felhasználású.

A listaelemek rendezettek, változtathatók, és engedélyezik az értékek megkettőzését.

  • Kódban:

    thislist = ["apple", "banana", "cherry"]

    print(thislist)

Feltételek

Az "if utasítás" az if kulcsszóval írható.

Az elif kulcsszó a pythons kifejezés, hogy "ha az előző feltételek nem voltak igazak, akkor próbálja ki ezt a feltételt".

Az else kulcsszó arra jó, amit az előző feltételek nem értek el.

  • Kódban:

    a = 200

    b = 33

    if b > a:

    print("b is greater than a")

    elif a == b:

    print("a and b are equal")

    else:

    print("a is greater than b")

Változók

A változók – számok, betűk, stb. – azon alapvető elemek közé tartoznak, amelyekkel a programok a működésük során dolgoznak.

A sztringek az str osztályba, az egész számok az int osztályba, a tizedespontot tartalmazó számok a float osztályba tartoznak.

A változó abban a pillanatban jön létre, amikor először értéket rendel hozzá.

  • Kódban:

    x = 5

    y = "John"

    print(x)

    print(y)
